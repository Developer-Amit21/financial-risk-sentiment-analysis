# 📊 Financial Risk & Sentiment Analysis

A data science project focused on **multi-class financial risk classification** and **sentiment analysis of news data**.
The system leverages machine learning and topic modeling to analyze financial news and provide **risk-aware insights** through an interactive dashboard.

---

## 🚀 Overview

This project combines **NLP, machine learning, and data visualization** to evaluate financial news and classify it into different risk categories.
It also identifies underlying topics and trends to support better decision-making.

---

## 🎯 Key Features

* 🧠 Multi-class **risk classification model**
* 📈 Financial **sentiment analysis**
* 📰 News-based **risk scoring system**
* 🧩 Topic modeling using BERTopic
* 📊 Interactive dashboard using Streamlit
* 🌍 Analysis of news trends and distribution

---

## 🛠️ Tech Stack

* Python
* Scikit-Learn
* XGBoost
* BERTopic
* Streamlit
* Pandas, NumPy, Matplotlib / Plotly

---

## ⚙️ How It Works

1. Collect financial news data
2. Preprocess text (cleaning, tokenization)
3. Perform sentiment analysis
4. Extract topics using BERTopic
5. Train a multi-class classification model
6. Predict risk levels for new inputs
7. Visualize insights via Streamlit dashboard

## ▶️ Run Locally


